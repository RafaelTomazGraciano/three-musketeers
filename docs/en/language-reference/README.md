## Quick Reference

### Basic Types
- `int` - Signed integers
- `double` - Floating point numbers
- `char` - Single characters
- `bool` - Boolean values (true/false)
- `string` - String literals

### Control Flow
- `if/else` - Conditional execution
- `switch/case` - Multi-way branching
- `for` - Counted loops
- `while` - Condition-based loops
- `do-while` - Execute-then-check loops
- `break` - Exit loops/switches
- `continue` - Skip loop iteration

### I/O Functions
- `printf` - Formatted output
- `scanf` - Formatted input
- `puts` - String output
- `gets` - String input

### Type Conversion
- `atoi` - String to int
- `atod` - String to double
- `itoa` - Int to string
- `dtoa` - Double to string
