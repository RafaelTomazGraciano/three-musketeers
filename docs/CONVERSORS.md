# Conversores de strings e dados
Esta seção irá tratar como fazer a conversão de tipos entre strings e outras e vice-versa

## Conversões de string para tipo x
- Para realizar a conversão de string para inteiro use `atoi(variável do tipo string ou STRING)
- Para realizar a conversão de string para double use `atod(variável do tipo string ou STRING);`

## Conversões de tipo x para string
- Para realizar a conversão de inteiro para string use `itoa(variável do tipo int ou INTEIRO);`
- Para realizar a conversão de double para string use `dtoa(variável do tipo double ou DOUBLE);`
