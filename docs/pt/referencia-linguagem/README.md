## Referência Rápida

### Tipos Básicos
- `int` - Inteiros com sinal
- `double` - Números de ponto flutuante
- `char` - Caracteres únicos
- `bool` - Valores booleanos (true/false)
- `string` - Literais de string

### Fluxo de Controle
- `if/else` - Execução condicional
- `switch/case` - Ramificação multi-caminho
- `for` - Loops contados
- `while` - Loops baseados em condição
- `do-while` - Loops executar-depois-verificar
- `break` - Sair de loops/switches
- `continue` - Pular iteração do loop

### Funções de E/S
- `printf` - Saída formatada
- `scanf` - Entrada formatada
- `puts` - Saída de string
- `gets` - Entrada de string

### Conversão de Tipos
- `atoi` - String para int
- `atod` - String para double
- `itoa` - Int para string
- `dtoa` - Double para string
