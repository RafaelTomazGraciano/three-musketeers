## Filosofia da Linguagem

Three Musketeers é projetada como um subconjunto da linguagem de programação C com recursos adicionais. A linguagem é fortemente tipada com conversão de tipo implícita, o que significa que todas as variáveis devem ser declaradas com um tipo, e o compilador automaticamente lida com conversões de tipo quando necessário.

## Características Principais

- Linguagem fortemente tipada com conversão de tipo implícita
- Suporte nativo para tipos `string` e `bool`
- Suporte completo a ponteiros
- Estruturas e uniões
- Arrays multi-dimensionais
- Declarações de fluxo de controle abrangentes
- Gerenciamento dinâmico de memória
- Operações de E/S padrão
