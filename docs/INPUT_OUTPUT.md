# Entrada e Saída
Essa seção trata como adquirir e imprir dados na linguagem Three Musketeers

## Adquirir Dados
Existem duas formas para adquirir os dados sendo estas:
- scanf — Adquire um conjunto de dados de tipos variados 
    - Sintaxe: `scanf(args...);`
- gets — Lê uma unica string
    - Sintaxe: `gets(variável de tipo string);`

## Imprimir Dados
Existem duas formas para imprimir os dados sendo estas:
- printf - Imprime para a tela com um determinado formato, a tabela após a sintaxe mostra os formatadores de tipo
    - Sintaxe: `printf(STRING COM FORMATADORES, args...);`
    
    | Formatador | Tipo  |
    |------------|-------|
    |     %d     |  int  |
    |     %f     | double|
    |     %s     | string|
    |     %c     | char  |
- puts - Imprime uma variável do tipo string ou uma string
    - Sintaxe: `puts(STRING ou variável do tipo string)`
